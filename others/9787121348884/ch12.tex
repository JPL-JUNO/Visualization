\chapter{颜色使用\label{ch12}}
\section{使用颜色参数和颜色映射表}
颜色参数 color 有以下几种使用模式：
\begin{itemize}
    \item 模式 1：英文缩写模式的基本颜色
    \item 模式 2：Hex 模式的 \#RRGGBB 字符串
    \item 模式 3：HTML/CSS 模式的颜色名
    \item Decimal 模式的归一化到 [0,1] 的 (R,G,B) 元组 color =(0.5294,0.8078,0.9216)
\end{itemize}
